\## Quick Study Cheat Sheet



\### \*\*Economics\*\*

\- \*\*Supply \& Demand\*\*

&nbsp; - Law of Demand: ↑Price → ↓Quantity demanded  

&nbsp; - Law of Supply: ↑Price → ↑Quantity supplied

\- \*\*Key Formulas\*\*

&nbsp; - GDP (Expenditure Approach):  

&nbsp;   `GDP = C + I + G + (X - M)`  

&nbsp;   Where C = consumption, I = investment, G = government spending, X = exports, M = imports

&nbsp; - Inflation Rate:  

&nbsp;   `Inflation (%) = (CPI this year - CPI last year) / CPI last year × 100`

\- \*\*Market Structures\*\*

&nbsp; - Perfect Competition, Monopoly, Oligopoly, Monopolistic Competition



---



\### \*\*Mathematics\*\*

\- \*\*Algebra\*\*

&nbsp; - Quadratic Formula: `x = (-b ± √(b² - 4ac)) / 2a`

\- \*\*Statistics\*\*

&nbsp; - Mean: `μ = Σx / n`  

&nbsp; - Variance: `σ² = Σ(x - μ)² / n`  

&nbsp; - Standard Deviation: `σ = √σ²`

\- \*\*Probability\*\*

&nbsp; - P(A or B) = P(A) + P(B) - P(A and B)  

&nbsp; - P(A and B) = P(A) × P(B) (if independent)
